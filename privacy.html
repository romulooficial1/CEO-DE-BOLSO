Politica de privacidade

Coletamos dados de conta, dados do negocio, historico de perguntas e status de assinatura para operar o CEO de Bolso AI.

Quando Firebase estiver configurado, as informacoes sao armazenadas no Firestore por usuario autenticado. Dados enviados para a IA sao usados para gerar respostas personalizadas.

Pagamentos sao processados pelo Mercado Pago. Nao armazenamos dados completos de cartao.
