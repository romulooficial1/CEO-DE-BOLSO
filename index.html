FAQ

Perguntas frequentes

Preciso configurar API?

Para publicar com IA real, configure OPENAIAPIKEY. Sem chave, o app usa fallback seguro.

Os dados ficam salvos?

Sim. Com Firebase configurado, dados do negocio, historico, uso e premium ficam no Firestore.

Como funciona o premium?

A assinatura mensal usa Mercado Pago com external_reference do UID e webhook para ativar premium.

Posso instalar no celular?

Sim. O app tem manifest, tema escuro e service worker para instalacao como PWA.
